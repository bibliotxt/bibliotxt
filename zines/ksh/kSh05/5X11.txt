En este texto basare mis conocimientos y experiencias en uno de los  celulares
mas pupulares nokia 51XX.

Para clonar un telefono celular necesitaremos:

1) SYSTEM ID = Sistema de Identificacion
2) AREA CODE = Codigo de Area
3) NUMERO DE TELEFONO A SER CLONADO

Pero en que consiste el SYSTEM ID?

Es el sistema de identificacion de area donde te encuentras. El cual  consiste
valga la redundancia,en un numero de cinco digitos. Ej:12463

Una de  las maneras  mas faciles  de obtener  el system  ID es  hacerlo con la
programacion de un telefono que este ya en una red.

Nota: Cada prooveedor tendra un numero diferente para esa area.

Para  el codigo  de area,  creo que  no tengo  que explicarlo,  pero para  los
despistados es  un numero  de tres  digitos. Ej:  809 (el  cual corresponde  a
nuestro pais, Republica Dominicana)...


Una vez  obtenidas esta  informacion, pasamos  a la  parte de  programacion de
nuestro celular :) Here we go!!!


|=|=|=|=|=|=|=|=|=|=|=|=|=|
PROGRAMANDO NUESTRO CELULAR
|=|=|=|=|=|=|=|=|=|=|=|=|=|

Introducimos los siguientes  digitos *3001#12345# esto  nos llevara a  un menu
diferente  el  cual utiliza  el  proovedor del  telefono.  Si este  codigo  no
funciona es porque  el proovedor ha  cambiado el 12345,  sin embargo no  he de
procuparse mucho por ello pues hay otros metodos o caminos para llegar a el ;)

Una vez realizado lo anterior, aparece el menu donde estan todas las  opciones
y configuraciones:

NAM1 y NAM2 = Identificadores primarios y secundarios. Normalmente donde 
	      su numero es almacenado. 


|=|=|=|=|
SECURITY    = Es el codigo de seguridad,por defecto es 12345
|=|=|=|=|

|=|=|=|=|
EMERGENCY   = Numeros de emergencia (911).Contrario a la creencia popular, 
|=|=|=|=|     cambiar el numero de emergencia no quiere decir que puedas
	      hacer llamadas gratuitas, sin embargo si decides probar :p

|=|=|=|=|
SW VERSION  = Version del Soft del celular. Ej 5160, 5190, 2110 etc.
|=|=|=|=|

|=|=|=|=|
SERIAL NO.  = Numero serial. 풬o necesita explicacion o si?  ;)
|=|=|=|=|

|=|=|=|=|
TEST FIELD  = Prueba de campo, frecuencia. Se utiliza para ver si esta 
|=|=|=|=|     funcionando correctamente el equipo y el servicio.



Cada opcion representa una funcion diferente. Donde NAM1 y NAM2 contiene
la informacion basica del celular, como decia anteriormente su numero
almacenado (SID).


Home Soc   (Casa Soc) <=== es el numero de operario de sistema el cual le 
			   dice que servicio esta usando.

Own Number (Numero propio) <=== Aqui nos muestra un numero de diez 
				digitos; el area + numero telefono.

Alpha tag (etiqueta alfa) <=== En la mayoria de los casos donde aparece
                               el nombre del proveedor, el cual puedes 
			       cambiar por tu nombre o nick.

PSID:RSID lists <=== son las listas de PSID (Private System ID) y RSID
		     (Residential System ID). Claro esta estos servicios
	             no vienen incluido en el celular a menos que se lo 
		     especifiques a tu proveedor.

Change Defaults (Cambios por defecto) <=== Se los dejo de tarea para 
					   para que investiguen :p


Una vez aprendido todo lo anterior  y suponiendo que ya tienen cierto  dominio
sobre el  tema. Vamos  a ver  otro metodo  de como  programar su  nokia de una
manera mas sencilla y rapida ;)

Para entrar a NAM1 digite *#6391# o *#639#
NAM2 digita *#6391#

Una vez entrado estos codigos el celular mostrara numero del celular. Entramos
los 10 numeros, recuerdan ( area +  # de celular). Y presionamos OK. Luego  el
telefono mostrara el  System ID, entramos  el que tenemos  y le asiganamos  un
codigo de  cierre, ej:  12345#, el  ultimo valor  es el  codigo de cierre y el
signo # es el numero de lenguaje. Los numeros para lenguaje son 0= INGLES,  1=
FRANCES, 2=ESPANOL Y 3= PORTUGUES.

Felicitaciones, usted acaba de programar su celular ;)
