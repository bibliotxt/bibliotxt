SISTEMA CELULAR 
Es aquel sistema de telecomunicaciones que nos permite tener una comunicación
constante entre usuarios que se encuentran a cierta distancia, aun cuando
estos equipos se encuentren en movimiento, quiere decir que para sector debe
existir una antena que cumple la función de retransmisión para que toda la
información del usuario se registre. Se requiere una central telefónica del
sistema móvil.
Toda antena retransmisora cubre un área el cual gráficamente se asemeja a una
célula, es por ello que todas las comunicaciones realizadas por los equipos
portátiles se llaman comunicaciones celulares (comunicación de célula a célula).




COMPONENTES DEL SISTEMA CELULAR 

El Sistema celular debe poseer como mínimo una central telefónica, estaciones
base o repetidores y equipos portátiles. 

MTSO ( Oficina Central de Conmutación Telecomunicaciones Móviles)
MTX 
Central de telefonía celular el cual provee una gran cantidad de enlaces que
nos va a permitir una comunicación constante. Posee dos sectores:
Sector de Control y Conmutación como el registro y la interconexión con otros
sistemas. Posee otro sector que permite la conexión inalámbrica por Fibra
Óptica, Microondas a este se le indica como unidad de R.F. y posee moduladores
y de moduladores, secciones de transmisión como secciones de recepción.

Debe realizar las siguientes funciones :
-	Recibir la información de cada una de las estaciones base.
-	Controlar la sectorización de la información 
-	Registrar a todo usuario según ESN como numero telefónico.
-	Comunicar de usuario a usuario.
-	Permite la comunicación con otros sistemas.
-	Permite elegir el tipo de enlace ya sea por Fibra Óptica o por 
        Microondas, etc.
-	Supervisa el estado de cada uno de las celdas como de los equipos
        portátiles.
-	Posee un área de trabajo que va de 12 Km. a 30 Km. de radio.

Estación Base
También llamado  Estación Repetidora, esta antena posee sectores de trabajo y
por cada sector posee circuitos independientes, como son los moduladores, 
demoduladores, retransmisores  y receptores para cada uno de los sectores.

FUNCIONES:
-	Permite controlar a cada uno de los equipos que se encuentran dentro
        del área de trabajo.
-	Toda información recibida se envía a la MTSO o a la central y esta a
        su vez envía una información de retorno.
-	La estación supervisa el estado de cada uno de los equipos celulares
        que se encuentren dentro de su área de trabajo.
-	Posee líneas de comunicación para datos y para voz.
-	La estación base supervisa a los equipos por medio de un canal de 
        control.
-	Las comunicaciones entre equipos celulares y estación base hacia otro
        equipo celular base realiza por medio del canal de voz (el canal de voz
        solo es usado cuando un equipo realiza o recibe llamadas).




CANAL DE CONTROL 
Este canal es usado para brindar una comunicación entre el equipo celular y la
estación base para que de esta manera se pueda supervisar el estado de cada uno
de los equipos dentro del sistema.
Este canal posee ciertas cualidades que permite supervisar simultáneamente a
varios equipos que s encuentren en reposo iniciando una llamada o recibiendo la
información del timbrado. Para esto solo se utiliza 21 canales de control.

CANAL DE VOZ
Este canal es usado como conducto de comunicación entre usuarios y solo se
utilizara una por cada comunicación para esto el sistema posee 395 canales que 
son exclusivos para la voz para el sistema analógico AMPS y 1185 canales para el
sistema NAMPS.

MU ( Unidad Móvil)
Es el equipo celular que nos va a permitir trabajar correctamente teniendo los
bloques de control y RF, para esto el equipo celular debe poseer una secuencia
de programa o funcionamiento lo cual nos ayudara a usar una gran cantidad de
funciones tales como vibrador, timbrado, atención automática, calculadora, 
sistema de mensajes cortos, hora y fecha, etc.
Para esto se va a realizar funciones internas con la ayuda de la memoria EEPROM.

FUNCIONES QUE REALIZA EL EQUIPO CELULAR
Todo equipo debe de enviar la información de su serie electrónica con un 
determinado numero telefonico hacia la estacion base desde el momento en que se
enciende el equipo. El equipo luego de registrarse debe de buscar una estacion
base con un canal de control disponible deacuerdo a la señal de calidad de señal
de recepción.
En el preciso momento en que apretamos "SEND"
etc, los siguientes datos son eviados a la celula. Estos datos son los
siguiente:

		ESN 
		codigo de area				
		MIN 
		numero al que llamamos


Luego, el switch, verifica que seamos abonados y estemos activos,
luego lo que hace es asignarnos un canal de voz libre.



Sistema Celular Namps 

Hoy en dia el sistema le proporciona identificación de numero de llamada
(Clip).
Pueden tener acceso a mensajes de voz que pueden ser almacenadas en la
central telefónica.
Trabajan normalmente con frecuencias para 2 bandas entre 824.04 Mz. A 
848.97 Mz. Para la transmisión y 869.04 Mz. Hasta 89.97 Mz. Para la 
recepción, teniendo una separación  de transmisión a recepción de 45 Mz.
Este sistema usa una transmisión constante los cual agota la bateria en 8
a 16 horas de funcionamiento. 
Cada canal de control supervisa a los usuarios del area de servicio.
Este sistema fue el primero en desarrollarse como estándar el cual es 
usado hasta dias por muchos usuarios.

3. Sistemas Inalámbricos

Sistema Trunking (sistema Troncalizado)
el sistema troncalizado se caracteriza por ser el primer servicio de radio 
eléctrico que concentra enlaces en una estación principal desde estaciones
retransmisoras.
Este sistema de radio tiene una comunicación half duplex normalmente, quiere
 decir que se comunica en un solo sentido por vez pero si se elige una 
comunicación con el sistema publico o con el sistema celular
esta comunicación hará uso de la privacidad el cual es en full duplex.
Toda la comunicación de radio puede ser realizada en forma individual o en 
forma grupal ( la comunicación se realiza por medio de un perifoneo en donde 
cualquier usuario que escuche puede atender el mensaje).

Transmisión por mensajes
Este método de transmisión permite tener una comunicación manteniendo el uso 
del canal aun cuando este no se encuentre transmitiendo. Este tipo de 
transmisión se parece al sistema celular .Este método de trasmisión puede 
causar congestión en los canales usados por el sistema de radio.

Transmisión Pura
Este método de transmisión permite maximizar el uso de canales disponibles 
para este sistema. Permite reutilizar los tiempos muertos o de no transmisión
permitiendo de esta manera ampliar varias veces la capacidad del sistema.

Transmisión Mixta
Este método de transmisión es el mas usado ya que puede variar en transmisión 
por mensajes a transmisión pura en función del trafico existente.

5. Definiciones

Nam (Modulo de Asignación Numérica)
Posee un conjunto de datos que permiten el funcionamiento de los equipos celulares. 
Todo equipo celular posee un Nam y el limite máximo es de cuatro. Contiene los 
datos propios del usuario para cada numero asignado.

ESN (Numero de serie Electrónico)
Es un dato único por cada equipo el cual no debe repetirse a nivel mundial. Este 
código es el único que identifica al equipo dentro de una empresa para un 
determinado propietario. Este valor es necesario en el momento de registro del equipo.

MIN (Numero de Identificación  Móvil) 
Este código identifica el área de trabajo, tal como la empresa para la cual esta 
trabajando, sin este código el equipo no tiene servicio. Normalmente es acompañado 
del numero telefónico, los códigos pueden ser  511 o 111 para telefónica en Lima y
716 para Bellsouth en Lima.

SID (Identificación del Sistema)
Este código va a identificar el país y la empresa en la cual se esta trabajando. Los
valores posibles son 32511 para telefónica en el Perú dentro de la ciudad de Lima y
2000 para Bellsouth en el Perú y para todos los departamentos.

ACCOLC (Clase de Acceso)
Este código nos permite determinar la categoría que posee el equipo, el cual esta 
acompañado del ultimo digito del numero telefónico. La prioridad seria con el digito
1 y la no prioridad con el Cero.

GIM (Grupo de Identificación Móvil)
Permite determinar la potencia máxima que va a usar el equipo celular, con un valor 
de 05, 10, 15.

IPCH (Primer Canal de Búsqueda)
Este valor va a determinar la banda para la cual se esta trabajando quiere decir que
el primer canal que va usar, y puede usarlo en números consecutivos superiores o 
inferiores a 333 para telefónica y 334 para Bellsouth.

Numero de Emergencia 
Son las llamadas que pueden realizarse aun cuando el equipo se encuentre bloqueado.

Canal de voz reverso : Nuevamente, 45 mhz menor que el V. CHannel

Switch:  Es la "computadora" de la celular que se encargado de enviar/recibir la 
informacion, asi tambiencomo de asignar los canales.
