que para usar auque fuera un simple editor de texto como el wordstar habia que

comandos del DOS que hace unos años era lo que se encontraba en todas las pcs.

se tenga algun nivel de conocimiento.

algunas cosillas mas.
Mucha de la informacion aqui expuesta es es para personas con un nivel medio 
en DOS y no se explicara a fono cada programa pues este no es un tutorial
de programacion de archivos *.BAT o de las propiedades del DOS, sino que 
basados en esos conocimientos bamos a trucar un poco su funcionamiento.

sacados del libro 1001 trucos para DOS y PC , pero todos los codigos fuente 
en lenguaje BATCHRUN son de mi autoria.



  
3.1 -INSTALARDOR DE CONTROLADOR DE DISPOSITIVO ANSI.SYS



Para poder hacer los dos siguientes trucos debes tener instalado el controlador de 
dispositivo ANSI.SYS.
Si no sabes si lo tienes intalado, debes cortar el siguiente codigo fuente en 
lenguaje BATCHRUN y gurdarlo con el nombre de INSTANSI.BAT. 
 
Este es codigo fuente que debes cortar ( ojo sin los puntos )

................................................................................
@ECHO OFF
FIND /C/I "DEVICE=C:\WINDOWS\COMMAND\ANSI.SYS" c:\CONFIG.SYS
IF ERRORLEVEL 0 IF NOT ERRORLEVEL 1 GOTO DOS
IF ERRORLEVEL 1 GOTO INI
:INI
ECHO DEVICE=C:\WINDOWS\COMMAND\ANSI.SYS >> C:\CONFIG.SYS 
GOTO DOS
:DOS
FIND /C/I "DEVICE=C:\WINDOWS\COMMAND\ANSI.SYS" c:\WINDOWS\CONFIG.SYS
IF ERRORLEVEL 0 IF NOT ERRORLEVEL 1 GOTO FIN
IF ERRORLEVEL 1 GOTO INICIO
:INICIO
ECHO DEVICE=C:\WINDOWS\COMMAND\ANSI.SYS >> C:\WINDOWS\CONFIG.SYS  
CLS
ECHO NO SE ENCUNTRA INTALADO EL CONTROLADOR DE DISPOSITIVO ANSI.SYS
ECHO PARA PODER TERMINAR DE INSTALARLO HAY QUE REINICIAR LA PC
ECHO SI DESEA REINICIAR SU PC AHORA MISMO OPRIMA CUALQUIER TECLA
ECHO SI NO LO DESEA , PULSE AL MISMO TIEMPO " Ctrl + C " PARA SALIR.
PAUSE > NUL
C:\WINDOWS\RUNDLL32.EXE USER.EXE,EXITWINDOWS
:FIN
CLS
ECHO YA SE ENCUNTRA INTALADO EL CONTROLADOR DE DISPOSITIVO ANSI.SYS
ECHO OPRIMA CUALQUIER TECLA PARA SALIR.
PAUSE > NUL
EXIT 
................................................................................

Ahora debes abrir tu MS-DOS y situarte en el directorio donde gurdaste el 
INSTANSI.BAT y telclear " INSTANSI " esta orden ejecutara el programa y te 
dira si lo tienes instalado o si lo deseas instalar.


NOTA: SI POR CASUALIDAD NO TE FUNCIONA ESTE INSTALADOR, PUEDE HABER SUCEDIDO QUE
LA ORDEN " DEVICE=C:\WINDOWS\COMMAND\ANSI.SYS " QUE SE SITUA EN EL FINAL DE LOS
ARCHIVOS CONFIG.SYS, HALLA QUEDADO EN EL MISMO REGLON DE OTRA ORDEN, LO UNICO
QUE TIENES QUE HACER PARA CORREGIR ESTE PROBLEMA ES DARLE ENTER PARA QUE QUEDE 
ESTA ORDEN SOLA EN UN RENGLON.

3.4 -SELECCIONAR EL COLOR DEL BORDE DE LA PANTALLA EN EL MODO TEXTO DEL MS-DOS


Para crear el programa que se encarga de realizar esta funcion debes cortar
el siguente codigo fuente de lenguje ensamblador y compilarlo mediante el uso
del debug que es una orden externa del MS-DOS que tienen todos los pc con este
sistema operativo.
Recorta el codigo que esta despues de estos puntos (ojo sin los puntos) y guardalo
con el nombre de BORDER.SCR

Este es codigo fuente que debes cortar ( ojo sin los puntos )
................................................................................ 
N BORDE.COM
A 100
MOV AX, 1001
MOV BH, 1 ;Este es el registro que debes modificar para cambiar de color
INT 10
MOV AH, 4C
INT 21

R CX
11
W
Q
................................................................................

Una vez lo tengas gurdado con el nombre de BORDER.SCR, abres el MS-DOS y te 
ubicas en el directorio en donde lo guardaste, ahora tecleas:

DEBUG < BORDER.SCR

y esto te creara el programa llamado BORDE.COM que al ejecutarlo te pondra el 
borde de tu pantalla de color azul.

Para que puedas ver que el programa realmente funciona tienes que tener la 
ventana de DOS del tamaño de toda la pantalla, para eso solo tienes que abrir 
tu DOS y teclear: ALT + ENTER y para volver al tamaño original ALT + ENTER
denuevo.
Si deseas elegir orto color solo tienes que modificar el registro BH en tu 
archivo BORDER.SCR y poner en vez de " 1 "el valor del color que quieras.

Esta es la tabla de valores:

0 = negro, 1 = azul, 2 = verde, 3 = azul turqueza, 4 = rojo, 5 = purpura,
6 = marron, 7 = blanco, 8 = gris oscuro, 9 = azul claro, A = verde claro,
B = azul tuqueza claro, C = rojo claro, D = purpura claro, E = amarillo,
F = blanco brillante.





3.5 -PROGRAMA PARA VER EL CODIGO RELATIVO DE UNA TECLA PULSADA.


Antes de comenzar me gustaria hacer enfasis en este programa ya que es probable
que para proximos numeros del ezine del C.C.H. vallamos a necesitar de este.
 
Para crear el programa que se encarga de ver el codigo relativo a una tecla 
pulsadadebes cortarel siguente codigo fuente de lenguje ensamblador y compilarlo 
mediante el uso del debug que es una orden externa del MS-DOS que tienen todos 
los pc con estesistema operativo.
Recorta el codigo que esta despues de estos puntos (ojo sin los puntos) y guardalo
con el nombre de CODTECLA.SCR
 
................................................................................
N CODTECLA.COM
A 100
INT 16
MOV AL, AH
MOV AH, 4C
INT 21

R CX
8
W
Q
................................................................................

Una vez lo tengas gurdado con el nombre de CODTECLA.SCR, abres el MS-DOS y te 
ubicas en el directorio en donde lo guardaste y tecleas:

DEBUG < CODTECLA.SCR

y esto te creara el programa llamado CODTECLA.COM el cual al ejecutarlo, solo 
finalizara al pulsar alguna tecla. Al hacer esto, la orden arrojara un codigo 
de salida relativo a la tecla pulsada.

Ahora empleando el programita que acabamos de crear vamos a hacer
un archivo *.BAT con el cual vamos a entenderer un poco mas su funcionamiento.
Recorta el siguiente codigo fuente en lenguaje de programacion BATCHRUN y guardalo
con el nombre de ASKEXIT.BAT en el directorio donde se encuentra CODTECLA.COM .

Este es codigo fuente que debes cortar ( ojo sin los puntos )
................................................................................
@echo off
cls
echo DESEAS SALIR DEL INTREPRETE DE COMANDOS DEL MS-DOS (S o N)?
CODTECLA
IF ERRORLEVEL 31 IF NOT ERRORLEVEL 32 GOTO SI
REM Este es el codigo de salida relativo a la tecla S
IF ERRORLEVEL 49 IF NOT ERRORLEVEL 50 GOTO NO
REM Este es el codigo de salida relativo a la tecla N
:SI
CLS

PAUSE > NUL
EXIT
:NO
CLS
ECHO BUENA ELECCION!  = )
PAUSE > NUL
CLS

la el programa CODTECLA.COM.
